苹果、安卓，还有 Windows 设备的隔阂终于被这软件打通了！ 原创 世超 差评 iCloud 的多设备同步功能有多好用想必正在使用苹果设备的差友应该都深有体会吧？ 只要你的 iPhone 开了 iCloud，手机里的照片、视频还有文稿就会自动同步到你所有登录 iCloud 账户的苹果设备中；反过来也是一样，其它设备上新增的文件也可以通过 iCloud 自动同步到 iPhone 上。 而且一旦你在其中一个设备修改了某个文件，那么其它设备上的这个文件也会自动更新为修改后的版本，这样一来每个设备上的文件就能一直保持在最新状态，这功能在日常使用的时候别提有多方便。 不过如果你没有苹果全家桶，而是手里只有一台苹果设备比如 iPhone、iPad 或者 Mac 的话。 那么抱歉，你是享受不到 iCloud 无缝同步功能带来的便利了。 这是因为 iCloud 到了 Windows 电脑上就是一个残废，不仅体验极差而且速度极慢，而且 iCloud 到目前为止根本就不支持 Andriod 设备。 这让目前正在使用安卓手机 + iPad + Windows 电脑组合的世超一度非常难受，平时我想要在这三台设备之间互传文件只能通过 QQ、微信等第三方软件来传，更别提能无缝同步文件了。 不过就在最近，世超发现有一个软件竟然早就实现了文件在 iPhone、iPad、Windows、Mac 还有 Android 之间的跨设备无缝同步，更关键是它在某些方面竟然做的比 iCloud 还好用你敢信？ 这款软件的名字叫微力同步，由于它有官方的使用说明文档，所以世超在这里就不介绍这软件具体怎么用了，我只给大家演示一下它到底有多强大。 因为这款软件我已经使用了好几天，所以差友们可以看到我在 Windows 电脑上早就建好了一个专门用来同步文件的文件夹，里面已经存在一些文件： 先打开我安卓手机上的微力同步 App，各位差友可以看到前面 Windows 电脑 “ 同步文件夹 ” 里的文件已经被这软件同步在手机里了： 这些文件你都能在自己手机上直接打开，或者在文件管理器里把它们拷贝其它位置： 到了 iPad 上也是一样，此时 Windows 电脑 “ 同步文件夹 ” 里的文件已经同步到了 iPad 中： 你也同样可以在 iPad 的文件 App 里直接对这些同步过来的文件进行操作： 如果我在 iPad 或者安卓手机里删掉其中一个文件： 那么很快，这个已经被 iPad 删掉的文件就会从手机还有电脑的 “ 同步文件夹 ” 里消失： 反过来也是一样，如果我在安卓手机里移一些照片到这个 “ 同步文件夹 ”： 那么这些照片用不了多久就会出现在 iPad 还有电脑的 “ 同步文件夹 ” 里： 至于像 PDF、Word 这类文档，微力同步跟 iCloud 一样，你在一个设备上编辑好这个 Word 文档并将它保存在 “ 同步文件夹 ” 后： 这个最新修改的 Word 就会同步在其它设备： 而且微力同步在同步文件夹的时候，不需要这些设备同处一个局域网，只要设备联网就行。 前面世超说的这些，还不是它最厉害的。 这款软件最厉害的地方是什么呢？ 在电脑端微力同步操作界面，你可以直接查看这些文件的修改记录，并且清楚地知道这些文件是在哪台设备上被修改的、它们的修改时间是什么时候： 你甚至还可以选择性的恢复这些文件的历史版本： 就算那些文件已经被你删掉，也还是会在本地缓存一段时间，直到你对它们进行手动清理，或者自定义一段时间让软件帮你自动清理，这些文件才会从本地正式消失，这个功能可以说是很人性化了： 反正世超这几天使用下来已经被这软件彻底折服了，在我看来，微力同步已经比较完美的解决了跨设备之间不能无缝同步文件的问题。 它甚至还支持 Linux 还有 NAS： 世超实测，目前安卓手机、Windows 还有 macOS 都可以免费下载这款软件。 而 iPhone 或者 iPad 用户如果要用这软件，则需要你花 12 块钱在 App Store 里下载，只要付一次费用，就能在这两个设备上同时使用。 世超猜测应该是软件上架到苹果商店的难度比较大，所以它才收费，正在使用iPhone 或者 iPad 的差友可以自行掂量这款软件值不值得你付费下载。 文章最后，微力同步的使用说明文档链接世超就放下面了，因为篇幅有限，软件有很多功能世超都没能跟各位细说，感兴趣的差友可以自己研究研究： http : //www.verysync.com/manual/introduction/ 感谢大家的观看！ 原标题：《苹果、安卓，还有 Windows 设备的隔阂终于被这软件打通了！》
